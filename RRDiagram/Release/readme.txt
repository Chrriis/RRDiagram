1. What is RR Diagram?

RR Diagram allows to generate railroad diagrams (also called syntax diagrams)
from code or from BNF notation. The output format is SVG which can be
integrated to web pages and where rules can contain links.

RR Diagram can also be used to generate BNF notation from a model.


2. How to use it?

Simply place the RRDiagram.jar library in your classpath.
Java 6.0 or later is required.
